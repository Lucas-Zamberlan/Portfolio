Habilidades

HTML →

Linguagem de marcação utilizada para estruturar o conteúdo da web. É fundamental para criar páginas web e definir elementos como texto, imagens e links

CSS →

CSS é uma linguagem usada para descrever a apresentação de um documento HTML. Define como elementos HTML devem ser exibidos, incluindo layout, cores, fontes, e outros aspectos visuais

JavaScript →

JavaScript é uma linguagem de programação que permite adicionar interatividade e comportamento dinâmico às páginas web. Com JavaScript, é possível criar funcionalidades como menus dropdown, sliders de imagens, validação de formulários, e outras interações do usuário

Python →

Python é uma linguagem de programação de alto nível, conhecida por sua simplicidade e legibilidade. É amplamente utilizada em diversas áreas, incluindo desenvolvimento web, ciência de dados, inteligência artificial, automação, e mais.

Git →

Git é um sistema de controle de versão distribuído usado para rastrear mudanças no código-fonte durante o desenvolvimento de software. Permite que vários desenvolvedores trabalhem no mesmo projeto simultaneamente, mantendo um histórico de todas as alterações.

GitHub →

GitHub é uma plataforma de hospedagem de código baseada na web que utiliza Git para controle de versão. Além de armazenar repositórios Git, GitHub oferece ferramentas para colaboração, como pull requests, issues, e wikis, facilitando o gerenciamento de projetos de software.

VBA →

VBA é uma versão do Visual Basic integrada ao Microsoft Office, permitindo automação e personalização de tarefas em aplicativos como Excel, Access, Word e Outlook. Com VBA, usuários podem criar macros, formulários personalizados, e scripts para automatizar processos repetitivos.

CLP →

CLP, ou PLC (Programmable Logic Controller) em inglês, é um computador digital robusto usado para automação de processos industriais, como controle de máquinas na linha de produção. CLPs são programados utilizando linguagens específicas, como ladder logic.

Arduino →

Arduino é uma plataforma de prototipagem eletrônica de código aberto baseada em hardware e software flexíveis e fáceis de usar. Utilizada por artistas, designers, engenheiros e amadores para criar projetos interativos.
